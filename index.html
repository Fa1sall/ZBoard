Rock, Paper & Scissors

The Rock Paper Scissors game project is a web-based application built using HTML, CSS, and JavaScript. It features a simple user interface with buttons for “Rock,” “Paper,” and “Scissors,” and displays the player’s choice, computer’s choice, and the result of each round. The game includes a scoreboard to track scores and uses JavaScript to handle game logic, such as random computer choices and determining the winner.
